Doors
=
A functioning version of the Doors for Garry's Mod.

Requirements
=
This addon requires the following:
- [world-portals](https://github.com/AmyJeanes/world-portals)

Without these the addon will not function correctly and you may experience errors.

Installation
=
### Steam Workshop
- [Doors (World Portals)](https://steamcommunity.com/sharedfiles/filedetails/?id=499280258) (release branch, stable)
- [Doors (World Portals) (Beta)](https://steamcommunity.com/sharedfiles/filedetails/?id=3574760963) (dev branch, less stable)

Note: [world-portals](https://github.com/AmyJeanes/world-portals) is bundled into the workshop addon for Doors already. It is not published standalone on workshop.

### Manual installation
Remove any previous versions then put this folder into Steam/Steamapps/common/garrysmod/garrysmod/addons/
